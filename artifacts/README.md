# artifacts

Эта директория предназначена только для воспроизводимых результатов выполнения ноутбуков.

Подкаталоги:

- `datasets/` — обработанные таблицы для следующих этапов;
- `metadata/` — сплиты и прочая служебная информация;
- `metrics/` — метрики моделей;
- `predictions/` — валидационные предсказания и сериализованные объекты;
- `params/` — сохраненные гиперпараметры;
- `submissions/` — финальные Kaggle-submission файлы;
- `reports/` — сводные таблицы для README и текста работы.
